## Getting Started

1. Clone the repository:
   ```bash
   git clone [your-repo-url]
   ```

2. Navigate to the client directory:
   ```bash
   cd client
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Run the development server:
   ```bash
   npm run dev
   ```

Visit `http://localhost:3002` to see the application.
